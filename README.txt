Metroblocks
by
Andrew Quintanilla
Makoa Fautua
Ray Liang

Python 3.2.3, pygame

Gameplay:
Form 2x2s of one solid color to be cleared by the wiper.

Controls:
x - Rotate block clockwise
z - Rotate block counterclockwise
Left arrow - Move block left
Right arrow - Move block right
Down arrow - Accelerate block down
